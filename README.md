## 安装
```
npm  i  -g  anydoor_tsuki
```
## 使用方法

```
tsuki  #把当前文件夹作为静态资源服务器根目录

tsuki  -p 8080 #设置端口 80 

tsuki -h localhost #设置 host  为localhost

tsuki -d /usr #设置根目录/usr
```


当然也可以github下载下来。 自己填写端口号. -p。 -h
npm install

node index -p 9999
